# 三角洲综合接单展示站

这是一个静态展示站，可以直接部署到 GitHub Pages。

## 本地预览

直接双击 `index.html` 即可打开。

## GitHub Pages 部署方式

1. 在 GitHub 新建一个仓库。
2. 把本项目推送到该仓库的 `main` 分支。
3. 打开 GitHub 仓库的 `Settings`。
4. 进入 `Pages`。
5. Source 选择 `GitHub Actions`。
6. 推送后等待 `Deploy static site to GitHub Pages` 工作流完成。

部署成功后，GitHub 会给出一个可公开访问的网址。

## 文件说明

- `index.html`：网站首页。
- `styles.css`：页面样式与响应式布局。
- `script.js`：复制联系方式、返回顶部等交互。
- `assets/hero-operations.png`：首页背景图。
